Purpose

The Most Traveled People iOS app.

Requirements

  • Xcode 9.4.1 or later
  • iOS 11.4 or later

Tools:

  • AppCenter for distribution management
  • Bundler for dependency management
  • CocoaPods for dependency management
  • Coveralls for test coverage statistics
  • Danger for automated code review
  • Slather) for test coverage reports
  • SwiftLint for opinionated coding style enforcement
  • SwiftyBeaver for logging
  • Travis-CI for test builds

Libraries:

  • Alamofire for HTTP networking
  • facebook-sdk-swift for Facebook support
  • Moya for network abstraction
  • R.swift for typed resource identifiers
  • Result for operation result modelling

Usage

Build the app with the ‘MTP’ target.
